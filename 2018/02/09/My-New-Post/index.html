变量的数据类型

  • 获取当前PHP消耗的内存 memory_get_usage
  • 定义字符串类型“$var_string = “123”;”
  • 定义整型“$n=123;”
  • 定义浮点型“$f=123.00;”
  • 定义数组“$var_array = array(“123”);”

PHP标量类型—布尔类型（boolean）

  • 只有两个值，一个是TRUE（是），另一个FALS（否）不分大小写
    -“var_dump”函数，表达数据类型。

####PHP标量类型—整型（integer）

  • 十进制 data_ = int10 ;
  • 八进制 data_ = int010; 数字前面必须加0
  • 十六进制 data_int = 0x10 ; 数字前面必须加0x

####PHP标量类型—浮点型

  • 通常说的小数，可以用小数点或者科学计数法表示
  • $num_float = 1.234; 小数点
  • $num_float = 1.2e3; 科学计数法，小写e 其中e3表示10的3次方
  • $num_float = 7.0E-10; //科学计数法，大写E 同理

####PHP标量类型-字符串

  • 字符串型可以用三种方法定义：单引号形式、双引号形式和Heredoc结构形式
    单引号、双引号、定界符。
    单引号：不解析转义字符，和变量。
    双引号：解析转义字符，和变量。
    定界符：都解析。
  • 单引号嵌双引号，双引号嵌单引号，直接写就行
    单引嵌单引或者双引嵌双引则要转义（加斜杠）
  • 当双引号中包含变量时，变量会与双引号中的内容连接在一起；
    当单引号中包含变量时，变量会被当做字符串输出。
  • 当字符串很长的时候 可以用<<<之后提供一个标识符 列如<<<d XXXX d 结尾的一行一定要另起一行
